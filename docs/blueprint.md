# **App Name**: Design Noir

## Core Features:

- Project Showcase: Display graphic design projects with high-quality images and descriptions, showing before/after images, and testimonials, when available.
- Category Filtering: Allow visitors to filter projects by category (e.g., branding, web design, illustration).
- AI Project Summarization: Employ an AI tool to provide concise, engaging summaries of each project, highlighting key design choices and outcomes.
- Contact Form: A simple form for potential clients to reach out directly.
- About Section: Detailed information about the graphic designer's experience, skills, and design philosophy.
- Testimonials Section: Showcase client testimonials to build trust and credibility.
- Skills Visualization: Visually represent the designer's skills and expertise (e.g., bar graphs for proficiency in different software).

## Style Guidelines:

- Primary color: Deep indigo (#4A148C), echoing the richness of graphic design while hinting at the Firebase color palette.
- Background color: Near-black (#121212), provides a stark contrast that makes design elements pop.
- Accent color: Electric violet (#8E24AA), drawn from the Firebase Studio palette, adds a modern, high-tech feel.
- Headline font: 'Space Grotesk', sans-serif, for headlines and short amounts of body text; body font: 'Inter', sans-serif, for longer text
- Use minimalist, geometric icons in a bright shade of electric violet (#8E24AA) for easy recognition and a modern feel.
- Emphasize whitespace to give the portfolio a clean, spacious layout. Use a grid system to align elements.
- Incorporate subtle fade-in animations for project elements and smooth transitions between sections.
